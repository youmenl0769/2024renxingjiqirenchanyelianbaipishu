 ## 产品趋势：体能与智能并举
 - 市场上的人形机器人可分为两大类型：
     - **体能型**：注重物理运动能力，适用于复杂环境操作。
         - **智能型**：专注于高级人工智能应用，包括决策制定和交互体验，更适合服务和教育等领域。
